## 如果你想体验我的项目，你需要准备：
#  1.（如果不想训练可以跳过此步）：
#    运行 check_gpu.py 来检查gpu环境是否配置好，如果版本不对，你需要下载安装适配你的GPU的pytorch（当然，cpu版本的也可以用，不过更慢）
#    以2026年4月我的RTX5060为例，你需要在终端输入下面这行代码 :
#    python -m pip install --pre torch torchvision torchaudio --index-url https://download.pytorch.org/whl/nightly/cu132
   
#  2.（如果不想训练可以跳过此步）：
#    运行 train.py 来训练模型，结果会保存到 Models/minesweeper_ddqn.pth
#    当然你也可以修改参数，默认是200局的胜率达到65%就换更难的地图参数，如果你想训练困难的地图，请把阈值改得低一点，如20%

#  3.
#    运行 practice.py 来测试模型的能力，默认是随机生成地图，0.5秒点开一步，难度需要你自己设置，如果是CNN模型则只能适配特定大小的扫雷。
